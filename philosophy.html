Philosophy

I like open-source, and I do agree with GNU's philosophy about free software, which has four essential freedoms for software:

  1. to run the program
  2. to study and change the program in source code form
  3. to redistribute exact copies
  4. to distribute modified versions

"Worst Case Scenario"

since the life became too easy, people started using social media apps too much, and everyone is watching "short videos" and using only smartphones, the young people (my generation) became lazy and not interested in computing anymore, and they can not focus in a long time, so those who addictive to TikToks and Shorts will be less likely to be a developer.
So in the future (in my opinion) there will be fewer developers specially for the "boring" technology like operating systems (Linux), low-level programming languages, and core developers who contribute to open source projects.

The Solution

Problem solving is fun, anyone can do anything with right education, anyone can be the next Linus Torvalds or Rasmus Lerdorf.
There's a lot of foundations like The Raspberry Pi Foundation which engage young people in learning computing and digital making skills through a thriving network of clubs and events.
